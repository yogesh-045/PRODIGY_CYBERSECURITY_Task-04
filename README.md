# PRODIGY_CYBERSECURITY_Task-04
Simple Keylogger
Task-04: Safe Input Logger (Ethical Demonstration)

This project demonstrates a safe and ethical input logging tool that records only the text typed inside the program itself.
It does NOT record system-wide keystrokes and does NOT run in the background, making it fully legal and safe for academic and internship purposes.
